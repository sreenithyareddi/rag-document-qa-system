# 📄 RAG Document QA System

An AI-powered document question-answering system built with Streamlit, ChromaDB, Sentence Transformers, and Groq.

## 🚀 Live Demo

[Launch the App](https://rag-document-app-system.streamlit.app/)

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://rag-document-app-system.streamlit.app/)

## Features

* Upload PDF, DOCX, and TXT files
* Semantic search with ChromaDB
* Question answering using Groq LLMs
* Interactive Streamlit interface

## Tech Stack

* Python
* Streamlit
* ChromaDB
* Sentence Transformers
* Groq API
